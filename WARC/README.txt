This WARC file contains one object. The object is a text string of the EICAR Anti-Malware Testfile. 

The testfile is a trigger that should force a "Virus Detected" response from anti-malware software. 

The testfile contains no active or malicious payload. 

Its usage is described here: http://www.eicar.org/86-0-Intended-use.html 

This WARC file is intended to test the AV setup for any repository that is aware of the WARC format, and purports to scan WARC containers for malicious content.
